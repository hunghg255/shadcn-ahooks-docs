## Examples

### Default Usage



<Preview name="useMount"
  registryDependencies={['useBoolean']}
 code={`
import useMount from '@/src/hooks/ahooks/useMount';
import useBoolean from '@/src/hooks/ahooks/useBoolean';

const MyComponent = () => {
  useMount(() => {
    console.info('mount');

    return () => {
      console.info('unmount');
    };
  });

  return <div>Hello World</div>;
};

const Example = () => {
  const [state, { toggle }] = useBoolean(false);

  return (
    <>
      <button type="button" onClick={toggle}>
        {state ? 'unmount' : 'mount'}
      </button>
      {state && <MyComponent />}
    </>
  );
};

export default Example;`} />



## API

```typescript
useMount(fn: EffectCallback);
```

### Params

| Property | Description                 | Type         | Default |
| -------- | --------------------------- | ------------ | ------- |
| fn       | The function to be executed | `EffectCallback` | -       |
