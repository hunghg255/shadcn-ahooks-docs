## API

```typescript
const isFocusWithin = useFocusWithin(
  target,
  {
   onFocus,
   onBlur,
   onChange
  }
);
```

### Params

| Property | Description        | Type                                                        | Default |
| -- | | | - | --- |
| isFocusWithin | Whether the focus is in the current area | `boolean` |
