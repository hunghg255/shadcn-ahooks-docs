## Examples

### Default usage

<Preview name="useSet"
 code={`
import useSet from '@/src/hooks/ahooks/useSet';


const Example = () => {
  const [set, { add, remove, reset }] = useSet(['Hello']);

  return (
    <div>
      <button type="button" onClick={() => add(String(Date.now()))}>
        Add Timestamp
      </button>
      <button
        type="button"
        onClick={() => remove('Hello')}
        disabled={!set.has('Hello')}
        style={{ margin: '0 8px' }}
      >
        Remove Hello
      </button>
      <button type="button" onClick={() => reset()}>
        Reset
      </button>
      <div style={{ marginTop: 16 }}>
        <pre>{JSON.stringify(Array.from(set), null, 2)}</pre>
      </div>
    </div>
  );
};

export default Example;`} />


## API

```typescript
const [
  set,
  {
    add,
    remove,
    reset
  }
] = useSet<K>(initialValue);
```

### Result

| Property | Description      | Type               |
| -------- | ---------------- | ------------------ |
| set      | Set object       | `Set<K>`           |
| add      | Add item         | `(key: K) => void` |
| remove   | Remove item      | `(key: K) => void` |
| reset    | Reset to default | `() => void`       |

### Params

| Property     | Description                 | Type          | Default |
| ------------ | --------------------------- | ------------- | ------- |
| initialValue | Optional, set default value | `Iterable<K>` | -       |
